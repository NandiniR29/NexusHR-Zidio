# NexusHR

## Enterprise Human Resource Management System

NexusHR is a Human Resource Management System (HRMS) developed using React, Spring Boot, and PostgreSQL. The system provides an efficient way to manage employee records, attendance, leave requests, payroll, documents, education, experience, bank details, and emergency contacts.

## Features

- Employee Management
- Attendance Management
- Leave Management
- Payroll Management
- Document Management
- Education Management
- Experience Management
- Bank Details Management
- Emergency Contact Management
- Secure Login using JWT Authentication

## Technology Stack

### Frontend
- React
- TypeScript
- Vite
- Material UI

### Backend
- Spring Boot
- Java
- Spring Security
- JWT Authentication

### Database
- PostgreSQL

## Project Structure

- Frontend: React + TypeScript
- Backend: Spring Boot REST API
- Database: PostgreSQL
